Política de Privacidad de MyFastCoach

Última actualización: 07/03/2025 En myFastCoach, respetamos y protegemos la privacidad de nuestros usuarios, por eso hemos desactivado la autenticación por email para un acceso libre y sin recopilar ningún tipo de información de nuestros usuarios. Al utilizar nuestra aplicación, usted acepta los términos de esta política de privacidad. Acerca de myFastCoach: MyFastCoach es una aplicación de deporte impulsada por la API de OpenAI, diseñada para optimizar el rendimiento de los usuarios mediante planes de entrenamiento personalizados. Información Recopilada Para ofrecer una experiencia personalizada y de calidad, recopilamos los siguientes datos: Datos Técnicos: Cookies Ejemplos de Uso de los Datos Los datos introducidos por el usuario se utilizan exclusivamente para mejorar la experiencia del usuario y generar recomendaciones personalizadas basadas en la información proporcionada. Algunos ejemplos incluyen: Generar planes de entrenamiento adaptados a la condición física y objetivos del usuario. Personalizar rutinas de ejercicio según el rendimiento y progresión del usuario. Proporcionar recordatorios sobre entrenamientos o desafíos deportivos. Procesar datos a través de la API de OpenAI para ofrecer asesoramiento en estrategias de mejora del rendimiento. 📌 Nota: Al aceptar esta política de privacidad, usted también acepta las políticas y términos de uso de la API de OpenAI. Uso de la Información No se recopila ningún tipo de información, MyFastCoach únicamente sirve de puente entre los servicios de OpenAI y la interfaz, de esta forma cumplimos con todas las normativas y regulaciones de recopilación de datos. Para ello, damos acceso librea todo el mundo sin necesidad de recopilar información. Uso de AdMob y Anuncios Personalizados Para ofrecer una experiencia optimizada, utilizamos AdMob para mostrar anuncios relevantes dentro de la aplicación. Los anuncios cumplen con las regulaciones de privacidad locales, como el GDPR en la Unión Europea y el CCPA en California. Los datos recopilados para personalizar anuncios no se vinculan directamente a la información personal del usuario. Seguridad de los Datos Implementamos medidas avanzadas para proteger su información: Encriptación de datos con el estándar AES-256. Monitoreo constante de los sistemas para prevenir accesos no autorizados. Los datos procesados por la API de OpenAI se manejan con estrictos protocolos de seguridad y confidencialidad. Cookies Utilizamos cookies estándar para mejorar la funcionalidad de la aplicación. Estas cookies cumplen con el Reglamento General de Protección de Datos (GDPR).
